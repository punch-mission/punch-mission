# Contributed Tools

These are analysis tools related to PUNCH data that are contributed by the scientific community at large. 
They are not developed or maintained by the SOC. 

**If you have PUNCH-related tools, please open an issue or pull request to get it added here.**

| Name  | Link |  Description | Language |
| ------------- | ------------- |  ---- | ----- | 
| SPD_MTM | [Paper](https://agupubs.onlinelibrary.wiley.com/doi/10.1029/2020JA028748), [Zenodo](https://doi.org/10.5281/zenodo.3703167) | Spectral analysis method for identifying periodic signals in geophysical time series | IDL |
| multitaper | [Paper](https://agupubs.onlinelibrary.wiley.com/doi/10.1029/2025JA034006), [GitHub](https://github.com/gaprieto/multitaper), [Example Jupyter Notebooks](https://github.com/hsalinasGIT/amtm_signaldetect)| Spectral analysis method for identifying periodic signals in geophysical time series, Python version of SPD_MTM | Python | 
| SynCOM | [Paper](https://iopscience.iop.org/article/10.3847/1538-4357/ad7a77), [Zenodo](https://zenodo.org/records/13941849) | An Empirical Model for High-resolution Simulations of Transient Solar Wind Flows | IDL|
